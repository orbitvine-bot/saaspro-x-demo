FONTS DIRECTORY - USAGE INFORMATION
====================================

This directory contains all font files used in the SaaSPro X template.

STRUCTURE:
├── Inter/                    # Inter font family
│   ├── Inter-Light.ttf      # Weight: 300
│   ├── Inter-Regular.ttf    # Weight: 400
│   ├── Inter-Medium.ttf     # Weight: 500
│   ├── Inter-SemiBold.ttf   # Weight: 600
│   ├── Inter-Bold.ttf       # Weight: 700
│   ├── Inter-ExtraBold.ttf  # Weight: 800
│   └── Inter-Black.ttf      # Weight: 900
│
├── Poppins/                 # Poppins font family
│   ├── Poppins-Regular.ttf  # Weight: 400
│   ├── Poppins-Medium.ttf   # Weight: 500
│   ├── Poppins-SemiBold.ttf # Weight: 600
│   ├── Poppins-Bold.ttf     # Weight: 700
│   └── Poppins-ExtraBold.ttf # Weight: 800
│
├── fa-solid-900.ttf         # Font Awesome Solid icons
└── fa-brands-400.ttf        # Font Awesome Brand icons

ATTRIBUTION REQUIREMENTS:
=========================

When using this template, you MUST include the following in your website's footer:

"Fonts: Inter & Poppins by Google Fonts, Icons: Font Awesome"

Or you can use the pre-formatted attribution in the template footer.

REPLACING FONTS:
================

To replace fonts:
1. Add new font files to this directory
2. Update @font-face declarations in index.html
3. Update CSS font-family variables in style.css
4. Update attribution in footer

IMPORTANT NOTES:
================

- All fonts are hosted locally for offline use
- No CDN dependencies = better privacy & reliability
- Font files are included in the template ZIP
- You must keep font attribution when using the template
- Do not redistribute font files separately from the template

For font licensing questions, refer to the individual font licenses.
